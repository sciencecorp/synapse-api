# Install 

### dependencies
for now, we use vcpkg to import other packages. However, soon we will use vcpkg to package this repo for external use.

- [cmake](https://cmake.org/)
- [vcpkg](https://learn.microsoft.com/en-us/vcpkg/get_started/overview)
    - fmt
    - gtest
    - grpc
    - protobuf

### build
```
make 
```

### Run
```
./build/libsynapse
```
